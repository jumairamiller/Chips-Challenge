Compiling Instructions:
Make sure the source folder has the following:
 - profiles.txt
 - level folder containing "[1 - 10].txt" level files.
 - src folder contining: 
	- all the .java files for the classes,
	- all the .fxml files for javafx displays,
	- and the folders img1 and img2 for the game's images.
	
If these are all present, you can compile the code and run the class Main to start the game.#
